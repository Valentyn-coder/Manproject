server.py - программа для головного серверу та телеграм боту. Для її запуску необхідно встановити програмні бібліотеки: peewee, pyTelegramBotAPI, schedule. Це необхідно зробити за допомогою команди pip install "ім'я пакету". Для того щоб це працювало необхідно встановити мову програмування Python 3.6+. Після запуску необхідно зайти у мессенджер Telegram та знайти бота за цим ім'ям: @Multitask_Air_Pollution_Bot 
 
device.py - программа для вимірюючего пристрою. Її можна запустити але вона не буде працювати через те, що була запущена не на вимірюючим пристрої який описаний в роботі ,а на звичайному компютері.

ManSite_vseukr - це папка в якій знаходиться код для сайту. Для її запуску необхідно встановити пакетний менеджер npm у директорію знаходження папки. Для цього необхідно в терміналі ввести ці команди:
cd client
npm i
npm start
Після введення команди npm start сайт запуститься через декілька хвилин.

Увага не всі функції можуть працювати корректно через те що в мене є тільки один вимірюючий пристрій наприклад функція "найближчі пристрої". А також пристрій може бути вимкнений в період тестувань программ.
